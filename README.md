# Проект 27_01_2025

## Проект 27_01_2025 это приложение для работы с банковскими картами, счами и данными клиента

## Установка:

1. Клонируйте репозиторий:
'''
get clone https://github.com/Valentina1764/BVG/tree/develop
'''
2. Установите зависимости:
'''
pip install  -r requirements.txt
'''

## Использование:
В консоле выполните команду:
'''
python main.py
'''

## Ввод данных:
Данные вводятся из файлов "card_account.txt" и "states.txt" в дериктории "Data"

## Вывод результатов
Результаты обработки выводятся в консоль
